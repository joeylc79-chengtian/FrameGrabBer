# 视频逐帧抽图工具 v0.9

视频逐帧抽图工具是一款离线视频抽帧软件，可以把 MP4、MOV、MKV、AVI 等常见视频导出为连续图片序列。软件支持批量导入、指定帧率、片段截取、低清预览视频和导出中停止，适合需要把视频拆成图片帧再交给其他系统播放或制作的工作流。

## 适用场景

- 无人机编队、矩阵屏幕式表演：将视频拆成连续图片，便于在不能直接播放原始视频的系统中逐帧播放。
- 舞台视觉、LED 屏、投影素材：快速生成图片序列，交给后续排播、合成或转换流程。
- 动画、动作参考：按原帧率或指定帧率抽取画面，方便逐帧检查动作节奏。
- 视频素材检查：截取指定时间段，生成图片或低清预览视频，快速确认画面内容。

## 主要功能

- 拖拽或点击添加视频，支持一次导入多个视频。
- 自动读取分辨率、帧率、时长和总帧数。
- 抽帧模式支持逐帧、每秒一帧、每几帧抽一帧。
- 逐帧模式下可指定目标帧率，默认常用值为 24fps。
- 批量导出时可统一设置帧率，也可为不同视频分别设置帧率。
- 可只导出指定视频片段，支持滑条调整和手动输入时间。
- 图片格式支持 PNG 和 JPG。
- 可选生成低清预览 MP4，用来检查图片序列播放节奏。
- 默认导出到视频所在文件夹，也可指定统一导出目录。
- 导出过程中可停止任务，未完成的临时文件会自动清理。
- 暗色 / 亮色 / 跟随系统三种界面模式。

## 使用方法

1. 运行软件。
2. 将一个或多个视频拖入窗口，或点击“添加视频”。
3. 检查左侧视频队列中的视频信息。
4. 在右侧设置抽帧模式、图片格式、导出目录等参数。
5. 如有需要，开启“指定帧率导出”或“只导出视频片段”。
6. 点击底部“开始导出”；导出过程中可点击“停止导出”中断任务。
7. 等待导出完成后，在输出文件夹中查看图片序列。

## 输出说明

每个视频都会生成独立的导出文件夹，不会和其他视频混在一起。文件夹名称会包含视频名、导出模式、导出时间和预计图片数量，便于整理和转交。

导出时会先写入临时目录，全部完成后才生成最终输出文件夹。如果用户中途停止导出或导出失败，未完成的临时文件会被清理。

如果开启“低清预览视频”，预览 MP4 会保存在对应图片序列文件夹内。

## 从源码运行

需要 Python 3.10 或更高版本。

```bash
pip install -r requirements.txt
python main.py
```

## 打包 EXE

Windows 下可运行：

```bat
build.bat
```

推荐发布文件夹版，启动速度比单文件版更快。当前打包后的文件名为：

```text
VideoFrameExtractorV0.9.exe
```

## 仓库文件说明

- `main.py`：程序入口。
- `app_bootstrap.py`：启动、图标和拖拽支持。
- `video_info.py`：读取视频基础信息。
- `export_plan.py`：计算抽帧计划。
- `export_runner.py`：执行导出任务、更新进度和处理中断清理。
- `ui_*.py`：界面模块。
- `requirements.txt`：运行依赖。
- `build.bat`：Windows 打包脚本。
- `VideoFrameExtractorV0.9.spec`：当前英文名打包配置。
