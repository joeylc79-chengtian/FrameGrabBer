# 视频逐帧抽图工具

> 离线独立 EXE · 自动识别帧率 · 支持拖拽

## 简介

一款轻量离线工具，将视频逐帧导出为图片。无需安装 Python 或 FFmpeg，**双击即用**。

- 🎬 支持 MP4 / AVI / MOV / MKV / FLV / WMV 等主流格式
- 🖱️ **拖拽支持**：直接把视频拉进窗口
- 🔍 **自动识别帧率**：30fps 视频按 30 帧/秒导出，60fps 按 60 帧/秒，不强制固定帧率
- ⚙️ 可设置导出间隔（逐帧 / 每秒1帧 / 自定义间隔）
- 📐 自动读取视频分辨率、时长、总帧数
- 💾 导出 PNG（无损）或 JPG（体积更小）

## 下载

前往 [Releases](../../releases) 页面，下载最新版 `视频逐帧抽图工具.exe`。

## 使用方法

1. 双击 `视频逐帧抽图工具.exe`
2. 将视频文件 **拖入窗口**，或点击「选择视频」
3. 等待自动读取视频信息（分辨率 / 帧率 / 总帧数）
4. 设置导出参数：
   - **格式**：PNG（无损）或 JPG（体积小）
   - **帧间隔**：逐帧 / 每秒1帧 / 自定义每 N 帧抽1帧
   - **文件前缀**：输出图片的命名前缀
5. 点击「开始抽图」，等待完成
6. 完成后可选择「打开输出文件夹」

## 输出位置

图片保存在 **视频同目录下以视频名命名的文件夹** 中。

例：`E:\视频\test.mp4` → 输出到 `E:\视频\test\`

## 注意事项

- 本工具 **完全离线运行**，无需联网
- Windows 系统直接运行，无需安装任何依赖
- 视频路径支持中文

## 技术说明

- 内置 `imageio-ffmpeg` 静态编译版 FFmpeg，零外部依赖
- 基于 Python + tkinter 构建，PyInstaller 打包为单文件 EXE
- 源码已上传，可自行修改重新打包
